# 三餐记食

一款健康饮食规划小程序，帮助用户管理菜品、创建饮食计划、追踪饮食习惯。

## 功能特性

- **菜品管理**：添加、编辑、删除菜品，支持分类、营养信息、适合餐次等属性
- **饮食计划**：创建周期性饮食计划，安排每日三餐
- **数据统计**：查看营养摄入情况，追踪饮食习惯
- **自动图片匹配**：根据菜品名称自动匹配默认图片

## 技术栈

- **框架**：uni-app (Vue 3)
- **样式**：SCSS + Design Tokens
- **图标**：IconPark
- **数据存储**：本地存储 (localStorage)

## 项目结构

```
├── pages/
│   ├── index/          # 首页 - 今日饮食计划
│   ├── dishes/         # 菜品管理
│   └── plan/           # 饮食计划
├── components/         # 公共组件
├── styles/             # 样式变量
├── utils/              # 工具函数
└── static/             # 静态资源
```

## 开发

1. 安装依赖
```bash
npm install
```

2. 使用 HBuilderX 运行到小程序模拟器
